<h1>Web Application</h1>

 ### [YouTube Demonstration](https://youtu.be/7eJexJVCqJo)

<h2>Description</h2>
A web application that will allow the user to access their data from any
device that has a browser. The software shall display in the format of a graph over time the number of hours spent on
a module per week. The ideal calculated number of hours should also be displayed on the
graph.
<br />


<h2>Languages and Utilities Used</h2>

- <b>C#</b> 
- <b>Java</b>
- <b>SQL Database</b>

<h2>Environments Used </h2>

- <b>Windows 10</b> (21H2)
